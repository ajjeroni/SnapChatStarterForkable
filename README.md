## About The Project
  The Snapchat Notes feature is a private space inside Snapchat where users can quickly jot down thoughts, reminders, or anything else they want to save for themselves. Unlike chats or stories, notes are not shared with anyone ‚Äî they‚Äôre personal and only visible to the user.

The goal is to make note-taking feel native to Snapchat: fast, lightweight, and mobile-first.
